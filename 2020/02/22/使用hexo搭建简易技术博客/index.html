简单整理一下使用hexo搭建技术博客的流程，日后再细化

  1. 安装node，以本机arch linux为例

  2. 通过npm安装hexo

  3. 初始化一个目录作为博客的工作目录

  4. 创建一篇博客文章

  5. 生成静态页面

  6. 运行服务器

  7. 进入/themes安装apollo主题及主题所需依赖

  8. 修改/vim _config.yml主题为apollo

  1. 安装git插件，同时修改/vim _config.yml中deploy的相关设置

  2. 安装hexo-admin，使可以通过web页面编写博客

    hexo server -d
